GUIA DE USO

Setup:
1.- Guiar cmd hasta la carpeta del proecto
2.- Introduce el siguiente comando sin las comillas: "pip install -r requirements.txt"


Ejecucion:
1.- Una vez instalado todas las librerias con el anterior comando, ejecuta "app.py"
2.- Despues de introducir tu nombre de usuario, saltara el ultimo mensaje enviado.
3.- Para enviar un mensaje simplemente escribelo en la consola y presiona enter
4.- A todas las personas conectadas al programa en el momento les saltara tu mensaje, pero en formato de 
	error, para que no se pueda identificar que es un mensaje
5.- El programa no se puede cerrar presionando ^C en el caso de ejecutarlo desde CMD, tendr√°s que cerrar
	la ventana
